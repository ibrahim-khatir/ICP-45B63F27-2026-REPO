# InternCareerPath Cloud Computing Internship

## Selected Projects

### Project 1: Static Website with CDN

#### Objective

Deploy a globally available static portfolio website using AWS S3 and CloudFront with HTTPS support.

#### Services Used

* Amazon S3
* Amazon CloudFront
* AWS Certificate Manager (ACM)

#### Expected Outcome

A secure static website accessible globally through a CDN.

---

### Project 2: Infrastructure as Code (Terraform)

#### Objective

Provision cloud infrastructure using Terraform instead of manual configuration.

#### Services Used

* Terraform
* Amazon S3
* Amazon CloudFront
* AWS IAM

#### Expected Outcome

Reusable and automated cloud infrastructure deployment.

## Reason for Selection

These projects provide practical experience with cloud storage, content delivery, security, automation, and Infrastructure as Code, which are essential skills for cloud engineering roles.
